# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | ✅ |
| 1.1.0   | ✅ |

## Reporting a Vulnerability

If you experience a problem, you can let us know by starting a [new issue](https://github.com/labcord/Ruvia/issues/new). Or you can send an e-mail to mdevelopmentstudio@hotmail.com.
